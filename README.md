### CocoaPods

Voodu is integrated with CocoaPods!

1. Add the following to your `Podfile`:

```
use_frameworks!
pod 'Voodu', '~> 1.0'
```

2. In your project directory, run `pod install`
3. Import the `Voodu` module wherever you need it
4. Profit
